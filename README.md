# oxipng

Multithreaded PNG optimizer written in Rust, now available as npm package

Based on [oxipng][oxipng] 4.0.3

---

## Installation

```sh
npm install oxipng
```

## Usage

```javascript
const { oxipngSync } = require('oxipng')

oxipngSync(['--help'])
```

## CLI

```sh
oxipng --help
```

[oxipng]: https://github.com/shssoichiro/oxipng
